Unit 17 — Absolute Value

The absolute value of a number is its distance from 0 on the number line.

|x|={xif x≥0−xif x<0

17.1 Basic Properties

  1. |a|≥0 (always nonnegative).
  2. |−a|=|a|.
  3. |ab|=|a||b|.
  4. |ab|=|a||b|,b≠0.
  5. Triangle inequality: |a+b|≤|a|+|b|.

17.2 Solving Absolute Value Equations

Case 1: |x|=a

  • If a≥0: x=a or x=−a.
  • If a<0: No solution.

Example 1
Solve |x|=7.

x=7orx=−7

Case 2: |x+b|=c

Split into two cases.

Example 2
Solve |x−3|=5.

x−3=5⇒x=8 x−3=−5⇒x=−2

17.3 Solving Absolute Value Inequalities

  1. |x|<a⇒−a<x<a
  2. |x|≤a⇒−a≤x≤a
  3. |x|>a⇒x<−a or x>a

Example 3
Solve |x−4|<3.

−3<x−4<3⇒1<x<7

Example 4
Solve |2x+1|≥5.

2x+1≥5⇒x≥2 2x+1≤−5⇒x≤−3

17.4 Absolute Value in Word Problems

Distance is always nonnegative.

Example 5
A number is within 4 units of 10. Find the range of possible values.

Equation: |x−10|≤4.

6≤x≤14

17.5 SAT Tips

  • Always split into cases.
  • Check for no solutions when RHS is negative.
  • Graph inequalities on number line if unsure.
  • Remember: absolute value = distance.

Practice Problems (Unit 17)

1) Solve |x|=12.
2) Solve |x+5|=0.
3) Solve |x−2|≤6.
4) Solve |3x+4|>10.
5) A number is within 2 units of −5. Write inequality and solution set.

Solutions

1) x=12 or x=−12.

2) x=−5.

3) −6≤x−2≤6⇒−4≤x≤8.

4) 3x+4>10⇒x>2
OR 3x+4<−10⇒x<−14/3.

5) |x+5|≤2⇒−7≤x≤−3.
