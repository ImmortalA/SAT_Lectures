Unit 24 — Statistics I

Unit 24 — Statistics I

Statistics involves describing and summarizing data.
On the SAT, you’ll be tested on measures of central tendency, spread, and data interpretation.

24.1 Mean, Median, and Mode

  • Mean (average):
Mean=Sum of valuesNumber of values
  • Median: Middle value when data is ordered.

  • If even number of values → average the two middle numbers.

  • Mode: The most frequent value.

Example 1
Data: [2,3,3,4,5]
- Mean =(2+3+3+4+5)/5=17/5=3.4
- Median =3
- Mode =3

24.2 Range and Standard Deviation

  • Range:
Range=Max−Min
  • Standard Deviation (SD):
    A measure of how spread out data is.
    Larger SD → more spread.
    Smaller SD → values closer to the mean.

Example 2
Set A: [10,10,10,10] → SD = 0 (no spread).
Set B: [5,10,15] → larger spread, larger SD.

24.3 Histograms and Dot Plots

  • Histogram: Shows frequency of values in intervals (bins).
  • Dot Plot: Uses dots above a number line to show frequency.

SAT often asks:
- Which interval is most common?
- Approximate median/mean from the graph.

24.4 Word Problems Involving Averages

Example 3
The average of 4 test scores is 80. If three of them are 85, 75, 90, find the missing score.

85+75+90+x4=80⇒250+x=320⇒x=70

24.5 SAT Tips

  • Always sort data before finding the median.
  • If one value is much larger/smaller → mean shifts, median less affected.
  • SD questions are usually relative (“which set has greater spread?”).
  • Carefully read graph axes.

Practice Problems (Unit 24)

1) Find the mean of [4,6,8,10].
2) Data: [2,5,7,8,12]. What is the median?
3) What is the mode of [1,2,2,3,4,4,4,5]?
4) Find the range of [9,14,22,5,17].
5) A histogram shows most students scored between 70–80. What does this tell you about the mode?

Solutions

1) Mean =(4+6+8+10)/4=28/4=7.

2) Ordered set [2,5,7,8,12] → median =7.

3) Mode =4 (appears 3 times).

4) Range =22−5=17.

5) The mode is in the 70–80 interval.
