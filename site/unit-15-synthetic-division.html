Unit 15 — Synthetic Division

Synthetic division is a shortcut for dividing polynomials when the divisor is of the form:

x−c

15.1 Performing Synthetic Division

Steps: 1. Write down coefficients of dividend (fill in 0 for missing terms).
2. Write c from (x−c).
3. Bring down the first coefficient.
4. Multiply by c, add to next coefficient, repeat.

Example 1
Divide 2x3+3x2−5x+6 by x−2.

Set up:

223−564141827924

Quotient: 2x2+7x+9, remainder 24.
So:

2x3+3x2−5x+6x−2=2x2+7x+9+24x−2

15.2 Equivalent Expressions

If remainder = 0, the divisor is a factor.

Example 2
Divide x3−6x2+11x−6 by x−1.

Set up:

11−611−61−561−560

Quotient: x2−5x+6, remainder 0.
So (x−1) is a factor.

15.3 The Remainder Theorem

For f(x) divided by (x−c), the remainder is simply f(c).

Example 3

f(x)=x3+2x2−3x+1

. Find remainder when divided by x−2

.

Compute f(2)=23+2(22)−3(2)+1=8+8−6+1=11.
So remainder = 11.

15.4 SAT Tips

  • Use remainder theorem to save time.
  • If f(c)=0, then (x−c) is a factor.
  • Synthetic division only works for divisors of form (x−c), not x2+...
