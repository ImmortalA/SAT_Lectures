Unit 13 — Functions

A function is a rule that assigns each input exactly one output.
Notation: f(x) means “the output of f when input is x.”

13.1 What is a Function?

  • Each x has only one y.
  • You cannot have two outputs for the same input.

Example 1
Is this a function?
- (1,2),(2,3),(3,4) → Yes (every input unique).
- (1,2),(1,3),(2,4) → No (input 1 has two outputs).

13.2 When is a Function Undefined?

  • Division by zero is not allowed.
  • Square roots of negatives are not real.
  • Logarithms of nonpositive numbers are undefined.

Example 2

f(x)=1x−3

is undefined at x=3

.

Example 3

g(x)=x−5

is undefined if x<5

.

13.3 Composite Functions

If f(x) and g(x) are functions, then:

(f∘g)(x)=f(g(x))

Example 4
If f(x)=2x+1 and g(x)=x2, find (f∘g)(3).

(f∘g)(3)=f(g(3))=f(9)=2(9)+1=19

13.4 Finding the Solutions to a Function

Solve f(x)=k for x.

Example 5
If f(x)=x2−4, solve f(x)=5.

x2−4=5⇒x2=9⇒x=±3

13.5 Identifying Function Graphs

  • Vertical Line Test: If a vertical line crosses graph more than once → not a function.

Example 6
- y=x2 → function (parabola passes test).
- Circle x2+y2=1 → not a function (fails vertical line test).

13.6 SAT Tips

  • Watch for undefined values.
  • Composition is a common SAT trick.
  • Always check if graphs pass vertical line test.
  • Solutions to f(x)=k mean x-values where graph crosses y=k.

Practice Problems (Unit 13)

1) Is y=x a function? Why or why not?
2) For h(x)=5x+2, where is it undefined?
3) If f(x)=3x−4 and g(x)=x2+1, find (g∘f)(2).
4) Solve f(x)=x2+2x when f(x)=8.
5) Does the graph of a sideways parabola x=y2 represent a function?

Solutions

1) Yes, y=x is a function since each input has only one output (though domain is x≥0).

2) Denominator =0 at x=−2, so undefined there.

3) (g∘f)(2)=g(f(2))=g(2)=(22+1)=5 where f(2)=3(2)−4=2. So g(2)=5.

4) x2+2x=8⇒x2+2x−8=0⇒(x+4)(x−2)=0⇒x=−4,2.

5) No, x=y2 fails vertical line test (two y for each x>0).
